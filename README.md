# Rust 🦀 WASM 🕸 Pong 🏓

A port of [javascript pong](https://codeincomplete.com/games/pong/) to rust.  Compiled to WebAssembly!

## Live Demo

[Play the live demo](https://pong.prawn.farm)

## How To

Build static assets:

```sh
sh build.sh
```

Run locally:

```sh
cargo web start --release
```
